# Installation
To install directly from GitHub use the devtools R package:

`install.packages("devtools")`
`library(devtools)`

`install_github("jjlynch2/OsteoShiny", ref = "v1.2.11")`

# Depends
* OsteoSort (>= 1.2.5)
* htmltools
* zip
* plyr
* DT
* shiny
